# 🚗 '자동차 경주' 기능 목록

### ✅ 진행 상황 확인

```
🔴 진행 전
🟢 진행 완료
🔵 테스트 완료
```

## ⚙️ 기능 목록

- 🟢 자동차 이름 입력
- 🔵 자동차 이름 유효성 검사
  - [x] 입력 공백 검사
  - [x] 자동차 이름 형식 검사
  - [x] 자동차 이름 개수 검사
  - [x] 자동차 이름 글자 수 검사
  - [x] 자동차 이름 중복 검사
- 🟢 경주 횟수 입력
- 🔵 경주 횟수 유효성 검사
  - [x] 입력 공백 검사
  - [x] 경주 횟수 형식 검사
- 🔵 랜덤 숫자 생성
- 🔴 횟수별 자동차 전진
- 🔴 횟수별 게임 결과 출력
- 🔴 최종 우승자 선발
- 🔴 최종 우승자 출력
